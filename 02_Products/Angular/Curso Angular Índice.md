## Instalar Angular CLI
```
npm install -g @angular/cli
```
si da problemas
``` bash
mkdir -p ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH="$HOME/.npm-global/bin:$PATH"
echo 'export PATH="$HOME/.npm-global/bin:$PATH"' >> ~/.bashrc  # O ~/.zshrc si usas zsh
source ~/.bashrc

```
